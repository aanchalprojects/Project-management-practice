# Project Performance Metrics (KPIs)

## Objective
To measure project performance against planned targets.

| Metric | Planned | Actual | Status |
|------|---------|--------|--------|
| Schedule Variance | 0 days | +1 day | Slight Delay |
| Cost Variance | ₹5,00,000 | ₹4,80,000 | Under Budget |
| Defect Rate | < 2% | 1.5% | Acceptable |
| Resource Utilization | 90% | 88% | Acceptable |

## Notes
KPIs are reviewed weekly to ensure project health.
